### 📌 Reverse Array (Recursion)

**Approach:**
- Swap first and last elements  
- Recursively solve for remaining array  

**Base Case:**
- Stop when start >= end  

**Complexity:**
- Time: O(n)  
- Space: O(n) (recursion stack)
