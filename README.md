📊 Sorting Algorithms in C (No Functions)

This repository contains implementations of basic sorting algorithms written in C, without using any user-defined functions.
The goal is to understand the core logic step-by-step by writing everything inside the "main()" function.

---

🚀 Implemented Algorithms

🫧 Bubble Sort

- Repeatedly compares adjacent elements
- Swaps them if they are in the wrong order
- After each iteration, the largest element moves to the end

Key Points:

- Simple and easy to understand
- Time Complexity: "O(n²)"
- Stable sorting algorithm

---

🎯 Selection Sort

- Finds the minimum element from the unsorted part
- Places it at the beginning
- Reduces unnecessary swaps

Key Points:

- Fewer swaps compared to Bubble Sort
- Time Complexity: "O(n²)"
- Not a stable sorting algorithm

---

- Focus on logic building
- Avoid confusion of function calls initially
- Strengthen fundamentals before modular coding
